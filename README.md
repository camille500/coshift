# Nova Delta

Website voor **Nova Delta** — AI-implementatie & adviesbureau. Gebouwd met Nuxt 3.

## Features

- Implementatie, consultancy en compliance voor AI
- Dashboards voor elk gebruik
- Privé-omgeving (geen training op uw data)
- Automatisering van alles wat u twee keer doet
- Volledig in het Nederlands
- Donker thema met glas-effecten en subtiele animaties

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run preview
```
